### 环境准备
确保安装了Python3.6+，pip，Flask，Requests,如果在mac上python的环境是3.6＋，安装Flask和Requests应使用pip3
```Python
pip3 install Flask==0.12.2 requests==2.18.4

```
与此同时还需要HTTTP客户端，比如Postman，cURL或其他客户端。
